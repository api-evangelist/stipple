# StippleThis is a repo for managing machine readable artifacts for the [Stipple APIs](https://stipple.com/api_docs). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Stipple APIs](https://stipple.com/api_docs).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://stipple.com/api_docs)- [Developer](https://stipple.com/api_docs)- [Twitter](https://twitter.com/stipple)- [GitHub](https://github.com/stipple)The [APIs.json](https://github.com/api-evangelist/stipple/blob/master/apis.json) index provides links to all the relevant links for the Stipple APIs, including the OpenAPI and Postman if they are present.
